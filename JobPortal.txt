Follow the step given below in order to run the project:
(1) First Download The Project.
(2) Extract Project.
(3) Copy Extracted Folder to c:\wamp\www folder.
(4) Now Open Phpmyadmin.
(5) Create New Database Named "job".
(6) Select Newly Created Database and Import Database File available in root directory of Project Folder.
(7) now you can run the project.
